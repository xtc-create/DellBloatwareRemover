# DellBloatwareRemover
# Dell & Waves Cleanup for Windows 11

A small PowerShell script to remove all the annoying preinstalled Dell apps and Waves Audio stuff from your Windows 11 machine. Stops services, uninstalls drivers, and nukes leftover folders so your PC runs cleaner.

---

## What it does

- Removes Dell Command | Update, SupportAssist, and other Dell bloatwares.  
- Stops and deletes Waves Audio services.  
- Cleans up leftover Waves folders and drivers from DriverStore.  

Basically, a “cleanup” for Dell pcs.

---

## How to use

1. Open PowerShell as admin.  
2. Run the script:  
```powershell
.\dell_waves_cleanup.ps1
```
